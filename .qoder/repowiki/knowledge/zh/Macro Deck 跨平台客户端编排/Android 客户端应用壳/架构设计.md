- 入口层：`MainActivity` 继承 `BridgeActivity`，在 `onCreate`/`onWindowFocusChanged` 中强制设置沉浸式全屏 UI 可见性标志，确保 WebView 全屏体验。
- 构建层：采用标准 Gradle 多模块结构，`app/build.gradle` 依赖 `capacitor-android` 与 `capacitor-cordova-android-plugins`，并通过 `variables.gradle` 统一管理 SDK 版本与依赖版本。
- 自动化层：`fastlane/Fastfile` 定义 `build` 与 `release` 车道，通过环境变量注入签名信息与 Play Store 凭证，实现从版本递增、AAB/APK 打包到上架发布的自动化闭环。
- 配置边界：`AndroidManifest.xml` 与 `res/` 资源目录定义应用基础元数据与 UI 资源，`proguard-rules.pro` 提供代码混淆规则，`google-services.json` 可选集成推送服务。
