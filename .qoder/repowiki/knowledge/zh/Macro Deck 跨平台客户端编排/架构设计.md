- **核心集成模式**：采用 `capacitor.config.ts` 作为单一事实来源，将 `src/` 中的 Angular Web 应用（`app` 模块）编译产物注入 `android/` 和 `ios/` 原生壳层。
- **自动化编排**：根目录 `.github/workflows/ci.yml` 定义了跨平台发布契约，通过 `reusable-*` 工作流串行触发 Web 基础构建、iOS/Android 并行原生编译及最终的多端制品归档。
- **配置同步**：利用 `ionic.config.json` 统一管理 Capacitor 与 Cordova 集成状态，确保 Web 逻辑与原生插件（如 SSL 处理、屏幕方向）在两端的一致性映射。
